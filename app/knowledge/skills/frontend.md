# Frontend Engineering Skills

## Languages
- **JavaScript / TypeScript** — core frontend languages
- **HTML / CSS** — structural and styling fundamentals

## Frameworks & Libraries
- **React** — component-based UI library; preferred for most new projects
- **Vue.js** — progressive framework; good for incremental adoption
- **Angular** — enterprise-grade framework with opinionated structure
- **Next.js** — React meta-framework for SSR/SSG

## Styling
- **CSS Modules / CSS-in-JS** — scoped styling solutions
- **Tailwind CSS** — utility-first CSS framework
- **Sass / SCSS** — CSS preprocessor with variables and nesting

## Build & Tooling
- **Vite / Webpack** — module bundlers
- **ESLint / Prettier** — linting and formatting

## Accessibility
- **WCAG 2.1** — Web Content Accessibility Guidelines
- **ARIA** — Accessible Rich Internet Applications attributes
