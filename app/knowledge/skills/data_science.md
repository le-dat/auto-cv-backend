# Data Science & Machine Learning Skills

## Languages & Libraries
- **Python** — primary language for ML and data analysis
- **R** — statistical computing and visualization
- **pandas** — data manipulation and analysis
- **NumPy** — numerical computing foundation
- **scikit-learn** — classical ML algorithms

## Deep Learning
- **PyTorch** — flexible deep learning framework
- **TensorFlow** — production-focused deep learning framework
- **Keras** — high-level neural networks API

## Data Engineering
- **Apache Spark** — distributed data processing
- **Airflow** — workflow orchestration
- **dbt** — data transformation in warehouse
- **Kafka** — event streaming platform

## ML Operations
- **MLflow** — ML lifecycle management
- ** Kubeflow** — ML toolkit for Kubernetes
- **Feature stores** — Feast, Tecton

## Statistics & Experimentation
- **A/B testing** — hypothesis-driven experimentation
- **Statistical significance** — p-values, confidence intervals
- **Feature engineering** — transforming raw data into model inputs

## Synonyms & equivalences
- ML = Machine Learning
- DL = Deep Learning
- ETL = Extract Transform Load
- DS = Data Science
